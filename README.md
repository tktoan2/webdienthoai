# Asp app + mysql 2019

1. Clone về

2. Chạy nhảy vào trong thư mục vừa mới tải về.

3. docker build -t imdbserver . 
  Sau bước này sẽ tạo ra 2 image là imdbserver và 1 image ko có tên chỉ có ID

4. docker run -it
